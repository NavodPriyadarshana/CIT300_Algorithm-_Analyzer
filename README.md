## Project Summary
This Java console application analyzes and compares execution time of different searching and sorting algorithms using arrays of sizes **100, 500, and 1000**.

The program measures performance of:
- Linear Search
- Binary Search
- Bubble Sort
- Quick Sort

The user can:
- Generate test arrays
- Enter target values to search
- Run each algorithm individually
- Compare execution results
